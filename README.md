# 🎵 音乐播放器

一个简单易用的 Web 音乐播放器，支持在手机和电脑上使用，**音乐文件会自动保存到浏览器本地存储，下次打开自动加载**。

## ✨ 功能特点

- ➕ **添加音乐文件**：支持添加多个音频文件
- 💾 **自动保存**：音乐文件自动保存到本地，刷新页面后依然存在
- 🔄 **顺序播放**：按照 1→2→3 的顺序播放
- 🔁 **循环播放**：最后一首播完后自动播放第一首
- ⏯️ **播放控制**：暂停/播放切换
- ⏭️ **曲目切换**：上一首/下一首
- ⏩ **快进功能**：快进 5 秒
- 🗑️ **清空列表**：一键清空所有音乐
- 📱 **PWA 支持**：可添加到手机主屏幕，像原生应用一样使用
- 🎨 **精美界面**：渐变色设计，支持移动端触摸

## 🚀 如何使用

### 在线访问（推荐）

直接访问：**https://braveheartforeverever-coder.github.io/music-player/**

### 在手机上使用

1. **访问网址**：用手机浏览器打开上面的链接

2. **添加到主屏幕**：
   - **iPhone (Safari)**：点击分享按钮 → "添加到主屏幕"
   - **Android (Chrome)**：点击菜单 ⋮ → "添加到主屏幕"

3. **添加音乐**：点击"添加音乐文件"，从手机中选择音频文件

4. **自动保存**：添加的音乐会自动保存，下次打开应用时会自动加载

## 🎮 控制说明

| 按钮 | 功能 |
|------|------|
| ➕ 添加音乐文件 | 从手机/电脑选择音频文件，自动保存到浏览器 |
| ▶️/⏸ | 播放/暂停 |
| ⏮ | 上一首（第一首时跳到最后一首）|
| ⏭ | 下一首（最后一首时跳到第一首）|
| +5s | 快进 5 秒 |
| 🗑️ 清空所有音乐 | 删除所有已保存的音乐文件 |

- 点击播放列表中的曲目可以直接跳转播放
- 点击进度条可以跳转到指定位置
- 音乐文件存储在浏览器本地，关闭网页后依然保留

## 🔧 技术栈

- HTML5 Audio API
- JavaScript (ES6+)
- IndexedDB（浏览器本地数据库）
- CSS3 (渐变、动画)
- PWA (Service Worker + Manifest)

## 📝 支持的音频格式

大多数现代浏览器支持以下格式：
- MP3
- WAV
- OGG
- M4A
- AAC

## 💾 数据存储说明

**音乐文件存储位置**：
- 使用 IndexedDB（浏览器本地数据库）
- 存储大小：通常 50MB - 几百 MB（取决于浏览器）
- **重要**：
  - ✅ 刷新页面后音乐依然存在
  - ✅ 关闭浏览器后音乐依然保留
  - ⚠️ 仅在当前设备/浏览器有效
  - ⚠️ 清除浏览器数据会丢失音乐
  - ⚠️ 换设备或浏览器需要重新添加

## 💡 提示

- 添加的音频文件会自动保存，无需手动操作
- 建议使用 Chrome、Safari、Firefox 等现代浏览器
- 在手机上使用时，确保音量已开启且未静音
- 如需清空所有音乐，点击"清空所有音乐"按钮

## 🐛 常见问题

**Q: 为什么刷新后音乐列表还在？**  
A: 应用使用 IndexedDB 自动保存音乐文件到浏览器本地，关闭网页后也会保留。

**Q: 如何删除已保存的音乐？**  
A: 点击播放列表下方的"清空所有音乐"按钮。

**Q: 音乐文件占用多少空间？**  
A: 取决于文件大小。浏览器通常允许存储 50MB - 几百 MB。

**Q: 换了手机/浏览器，音乐还在吗？**  
A: 不在。音乐只保存在当前设备的当前浏览器中。换设备需要重新添加。

**Q: 为什么某些音频文件无法播放？**  
A: 请确保文件格式被你的浏览器支持。建议使用 MP3 格式。

**Q: 如何在手机上使用？**  
A: 访问 https://braveheartforeverever-coder.github.io/music-player/，然后添加到主屏幕。
