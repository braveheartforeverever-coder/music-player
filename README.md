# 🎵 音乐播放器

一个简单易用的 Web 音乐播放器，支持在手机和电脑上使用。

## ✨ 功能特点

- ➕ **添加音乐文件**：支持添加多个音频文件
- 🔄 **顺序播放**：按照 1→2→3 的顺序播放
- 🔁 **循环播放**：最后一首播完后自动播放第一首
- ⏯️ **播放控制**：暂停/播放切换
- ⏭️ **曲目切换**：上一首/下一首
- ⏩ **快进功能**：快进 5 秒
- 📱 **PWA 支持**：可添加到手机主屏幕，像原生应用一样使用
- 🎨 **精美界面**：渐变色设计，支持移动端触摸

## 🚀 如何使用

### 在电脑上

1. 用浏览器打开 `index.html` 文件
2. 点击"添加音乐文件"按钮选择音频文件
3. 点击播放按钮开始播放

### 在手机上

#### 方法一：本地服务器（推荐）

1. 在项目目录下运行本地服务器：
   ```bash
   # 如果有 Python 3
   python3 -m http.server 8000
   
   # 如果有 Python 2
   python -m SimpleHTTPServer 8000
   
   # 如果有 Node.js
   npx http-server -p 8000
   ```

2. 在手机浏览器中访问：`http://你的电脑IP:8000`
   - 查看电脑 IP：
     - Mac: `ifconfig | grep inet`
     - Windows: `ipconfig`

3. 添加到主屏幕：
   - **iOS Safari**：点击分享按钮 → 添加到主屏幕
   - **Android Chrome**：点击菜单 → 添加到主屏幕

#### 方法二：直接打开文件

1. 将整个 `music-player` 文件夹传输到手机
2. 用手机浏览器打开 `index.html` 文件

## 🎮 控制说明

| 按钮 | 功能 |
|------|------|
| ➕ | 添加音乐文件 |
| ▶️/⏸ | 播放/暂停 |
| ⏮ | 上一首（第一首时跳到最后一首）|
| ⏭ | 下一首（最后一首时跳到第一首）|
| +5s | 快进 5 秒 |

点击播放列表中的曲目可以直接跳转播放。

点击进度条可以跳转到指定位置。

## 📝 支持的音频格式

大多数现代浏览器支持以下格式：
- MP3
- WAV
- OGG
- M4A
- AAC

## 💡 提示

- 添加的音频文件只在当前会话有效，刷新页面后需要重新添加
- 建议使用 Chrome、Safari、Firefox 等现代浏览器
- 在手机上使用时，确保音量已开启且未静音

## 🐛 常见问题

**Q: 为什么刷新后音乐列表消失了？**  
A: 由于浏览器安全限制，无法永久保存文件引用。每次使用需要重新添加音频文件。

**Q: 为什么某些音频文件无法播放？**  
A: 请确保文件格式被你的浏览器支持。建议使用 MP3 格式。

**Q: 如何在手机上使用？**  
A: 参考上方"在手机上"部分，推荐使用本地服务器方式。
